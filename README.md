# Result Management System
## Project Overview
The **Result Management System** is a web-based application designed to simplify the process of managing and publishing academic results.  
It allows administrators and teachers to manage student records and results efficiently.

This project was developed as part of a **university academic requirement**.

## Objectives
- Digitize the traditional result processing system
- Reduce manual effort and errors in result management
- Maintain organized academic records

## Core Features
- Create, update, delete, and read the student records
- Display the course-wise result sheet
- Download the result sheet in PDF
- Convert the marks in the corresponding CGPA

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP(CodeIgniter)  
- **Database:** MySQL  
- **Server:** Apache (XAMPP)
